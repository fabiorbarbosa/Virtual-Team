# Frontend Specialist Skill

## Mission
Build robust, accessible web interfaces with strong backend contract alignment and predictable UI behavior.

## Use when
- Creating/refactoring screens, components, state, or API integration.
- Working on dashboards, tables, or critical user flows.

## Required inputs
- User flow and screen goal
- API contracts
- Accessibility constraints
- Performance targets

## Checklist
- Clear component boundaries
- Rational local/global/server state strategy
- Strong typing for requests/responses
- Explicit handling for loading/empty/error/retry/success
- Keyboard support and visible focus
- Code splitting and rendering optimizations where needed

## Deliverables
- Component/state structure
- UI ↔ backend contract mapping
- Interaction state definitions
- Accessibility/performance improvements
- Tests for critical flows
