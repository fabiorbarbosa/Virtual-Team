# Mobile Specialist Skill

## Mission
Deliver stable, secure mobile flows with resilient offline/online behavior and platform-aware UX.

## Use when
- Building/refactoring mobile modules.
- Implementing sync, caching, permissions, notifications, or background work.

## Required inputs
- Critical user journeys
- Backend contracts
- Platform constraints
- Security requirements

## Checklist
- Modular architecture and clear state ownership
- Cache and sync policy
- Conflict resolution for offline updates
- Permission handling in the right context
- Lifecycle and background behavior
- Tests and production observability

## Deliverables
- Mobile architecture proposal
- Persistence/sync strategy
- Permission/background execution plan
- Core test package
