# DevOps Specialist Skill

## Mission
Ensure reliable operation, safe delivery, and fast recovery with observable, auditable pipelines and runtimes.

## Use when
- Creating CI/CD pipelines
- Containerizing apps
- Defining observability and deployment strategy
- Planning rollback and continuity

## Required inputs
- Service topology
- Dependencies
- Availability/RTO/RPO requirements
- Environment strategy
- Security constraints

## Checklist
- Minimal, secure runtime/container
- Deterministic pipeline stages
- Mandatory test/security gates
- Health checks
- Logs/metrics/traces/alerts
- Rollback and runbooks
- Backup/restore validation

## Deliverables
- CI/CD blueprint
- Runtime/container standards
- Observability plan
- Rollback and continuity strategy
