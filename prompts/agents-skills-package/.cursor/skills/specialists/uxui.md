# UX/UI Specialist Skill

## Mission
Design clear, low-friction journeys with explicit decision points, safe defaults, and strong recovery paths.

## Use when
- Designing or refactoring flows
- Reorganizing IA/navigation
- Reviewing dense screens or exceptional states

## Required inputs
- User goal
- Personas/context of use
- Regulatory communication constraints
- Success metrics

## Checklist
- Main path and alternatives documented
- Confirmation for irreversible actions
- Clear grouping by user goal
- Defined states: loading, empty, error, retry, success, partial failure
- Actionable error messages
- Reduced cognitive load and visible impact of decisions

## Deliverables
- User flow map
- IA guidance
- Interaction-state guidance
- Prioritized usability recommendations
